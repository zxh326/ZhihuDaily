API 说明

  • 消息以JSON格式输出
  • 可用接口如下:

[日报接口] daily: GET / POST[date=xxxx-xx-xx] localhost:5000:/api/daily 不可为空

[邮箱接口] Mail : GET / POST[subr=xxxx@xx.xx] localhost:5000:/api/mail, 不可为空

[日报今天] today: GET / POST localhost:5000:/api/daily/today, 无参

[日报随机] random: GET / POST [count = n] localhost:5000:/api/daily/random, 可为空默认1

[糗事百科] qiubai: GET / POST [page = n,count = n] localhost:5000/api/qiubai 可为空 1，10

使用详解

  • nextday : 下次提交时期
  • question : 问题列表,包含2-5个问题 不为空
    • answer:答案列
      • Author:答案作者
      • Author_bio：作者的一句话 / 可能为空
      • Author_image：作者头像
      • body：答案内容,有些可能过长
    • title:问题标题
    • url:问题原链接 ‘ ’ ‘
